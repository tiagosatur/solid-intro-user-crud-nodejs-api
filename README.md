# SOLID intro user CRUD NodeJs API

## :doughnut: About
This a user CRUD api built for nodejs study purposes

## :inbox_tray: Install
`yarn install`

## :runner: Run

`yarn dev`

The api will be available under the route `http://localhost:3333`

## :checkered_flag: Test

`yarn test`
